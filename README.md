# Produtos_API

3 tres codigos que consomem uma API por meio de uma URL isso em JavaScript.
 
Produtos - Documentos
O endpoint de produtos fornece um conjunto de dados abrangente com informações de produtos de amostra, incluindo detalhes como nomes, preços, descrições, imagens e categorias, ideal para testar e criar protótipos de aplicativos de comércio eletrônico.
